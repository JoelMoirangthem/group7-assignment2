Matrix Operations

This application performs various matrix operations such as generating a matrix, performing elimination, creating a diagonal matrix, calculating upper matrices, and decomposing matrices into LDU format. Follow the steps to complete each matrix operation.

Specify the order of the matrix (n x n), for instance, enter "3" for a 3x3 matrix. This input will determine the dimensions of all matrices created in the operations below.

Matrix A

In this section, you will create the initial matrix A based on the specified order. This matrix will serve as the foundation for subsequent operations, including elimination and decomposition.

Elimination Steps & U

This step performs Gaussian elimination on matrix A to reduce it to an upper triangular form, resulting in the matrix U. The elimination steps and the resultant U matrix are displayed here.

Diagonal Matrix D

Here, we create a diagonal matrix D from matrix U. Each diagonal entry reflects the scaling factors applied during the elimination process.

U' Matrix

U' matrix is calculated by normalizing the upper triangular matrix U using the diagonal elements of D.

L Matrix

The L matrix is derived from the elimination matrices applied to transform A into U. It represents the lower triangular matrix in the decomposition process.

LDU Decomposition

This is the final LDU decomposition of the original matrix A. It is the product of the matrices L, D, and U', which reconstruct A.
